# Beyond Culture Site

First-pass static landing page for `www.beyondculture.io`.

## Run locally

```bash
cd /root/.openclaw/workspace/beyondculture-site
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Notes

- Single-file static site for fast iteration and easy deployment
